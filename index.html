Card Sort Tool

A browser-based tool for running open, hybrid, and closed card sort studies.

Features

Three sort types are supported. The method is selected when the study is configured.

Open

Open card sort

Participants create their own categories. No predefined groups are provided.

Hybrid

Hybrid card sort

Starter categories are provided, and participants may add their own.

Closed

Closed card sort

Participants sort cards into a fixed set of predefined categories. New categories cannot be added.

groups

Multi-admin studies

Additional researchers can join an existing study using its study ID and study key.

shuffle

Randomized order

Card order is randomized per participant.

drag_indicator

Drag-and-drop interface

The participant interface runs in the browser on desktop and tablet. No installation is required.

lock

Row-level security

Researchers can read and write data only for studies they own or have joined.

About

The Card Sort Tool is a browser-based application for running card sort studies. Researchers create studies in the admin dashboard, share a participant link, and review submissions and analysis in the same dashboard.

The application is a static site built with HTML, CSS, and vanilla JavaScript. It uses Supabase for authentication and data storage, Google Charts for visualizations, and jsPDF with html2canvas for PDF export.
